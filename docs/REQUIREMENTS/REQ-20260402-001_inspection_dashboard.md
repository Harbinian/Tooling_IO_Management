## 5W2H 分析 / 5W2H Analysis

### What - 核心需求

**功能描述**：

工装定检任务管理组件的配套前端增强功能，包含三个独立视图：

1. **工装定检状态看板** (`InspectionDashboard.vue`)
   - 四宫格统计：待检数量、进行中数量、已逾期数量、已完成数量
   - 工装定检状态表格：展示所有工装的定检状态（序列号、工装名称、最近定检日期、下次定检日期、定检周期、状态）
   - 多维度筛选：按部门（org_id）、状态（normal/overdue/pending）、到期时间范围筛选
   - 支持导出 Excel

2. **定检任务统计面板** (`InspectionStats.vue`)
   - 任务数量统计：各状态（pending/received/in_progress/report_submitted/accepted/closed）的任务数量分布
   - 逾期率统计：任务逾期率、平均处理时长、各班组长完成率排名
   - 周期对比：本月 vs 上月 任务数量和逾期率对比
   - 时间范围筛选：默认最近30天，支持自定义日期范围选择器
   - 图表展示：柱状图（任务数量分布）、折线图（逾期趋势）

3. **定检日历视图** (`InspectionCalendar.vue`)
   - 月/周视图切换
   - 每天格子中显示任务数量，点击弹出任务详情卡片
   - **拖拽调整**：计划员可拖拽任务卡片调整计划时间，自动同步更新后端
   - 颜色编码：按状态显示不同颜色（pending=灰、in_progress=蓝、overdue=红、closed=绿）

**输入**：
- 工装定检状态数据（来自 `tool_io_tool_inspection_status` 表）
- 定检任务数据（来自 `tool_io_inspection_task` 表）
- 定检任务管理数据（来自 `tool_io_inspection_plan` 表）

**输出**：
- 统计卡片数字
- 筛选后的表格数据
- 日历可视化
- 导出文件（Excel）

---

### Why - 动机与价值

**业务背景**：
- 定检任务管理组件上线后，用户需要全局视角查看定检执行状态
- 计划员需要快速了解哪些工装即将到期需要安排定检
- 管理层需要统计数据辅助决策

**预期价值**：
- 提升计划员工作效率（一眼看清全局状态）
- 辅助质量管控（逾期率统计）
- 优化资源调配（周期对比）

**成功标准**：
- 看板加载时间 < 2s
- 筛选操作响应时间 < 500ms
- 日历拖拽操作后数据立即同步更新

### Who - 角色与用户

**主要用户**：
| 角色 | 可访问功能 |
|------|-----------|
| PLANNER（物资保障部计划员） | 全部三个视图，可拖拽调整日历 |
| KEEPER（保管员） | 看板（自己管辖工装）、统计面板、日历 |
| TEAM_LEADER（运维安环部班组长） | 看板（自己执行的任务）、统计面板 |
| SYS_ADMIN | 全部三个视图，无数据范围限制 |

**数据范围**：
- 按 org_id 过滤：不同组织用户只能看到本组织的工装/任务
- TEAM_LEADER 按 assigned_to_id 过滤
- SYS_ADMIN 无限制

### How - 实现方式

**建议实现方式**：
- 前端独立页面组件，挂在 `/inspection/` 路由下
- 图表库：ECharts（轻量、Vue 集成良好）
- 日历组件：`v-calendar` 或 `@fullcalendar/vue`
- 拖拽功能：Vue Draggable 或原生 Drag API

**参考系统**：
- 现有 DashboardOverview.vue（结构参考）
- Element Plus 统计卡片组件

**技术栈**：Vue 3 + Element Plus + ECharts + Vite

**导航结构约束**（来自用户补充需求）：
- 工装管理系统（一级入口）
  - 出入库（二级菜单）：订单列表、创建申请、保管员工作台、预知运输
  - 定检任务管理（二级菜单）：计划列表、任务列表、定检看板、定检统计、定检日历、工装定检状态

**风格约束**（来自用户补充需求）：
- 所有页面必须与现有工装出入库界面（OrderCreate.vue、OrderList.vue、OrderDetail.vue）保持一致
- Card 布局、Header 区域、Badge 标签、状态标签统一使用现有样式模式
- 复用 `WorkflowStepper` 组件（由 00084 提供）
