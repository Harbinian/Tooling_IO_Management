# 工装出入库管理系统 - 移动端帮助文档

## 系统概述

工装出入库管理系统（Tooling IO Management System）用于管理工装的出库和入库生命周期，确保工装在生产过程中的流转可追踪、可控制。

**访问地址**: http://localhost:8150 (前端)

---

### 班组长 (Team Leader)

::: success 班组长标签
**核心职责**: 创建订单、出库最终确认
**数据范围**: 本人 + 本组织
:::

**可执行操作**:
- 📋 创建出库单

- 🔍 搜索工装库存
- 📄 查看自己发起的订单
- ✅ 确认出库完成
- ❌ 取消自己发起的订单

**权限说明**:
| 权限 | 说明 |
|------|------|
| `order:create` | 创建订单 |
| `order:submit` | 提交订单 |
| `order:final_confirm` | 出库最终确认 |
| `order:cancel` | 取消订单 |
| `tool:search` | 工装搜索 |

## 业务流程

### 出库流程

```
┌─────────┐    ┌──────────┐    ┌──────────┐    ┌────────┐    ┌─────────┐
│ 草稿    │ → │ 已提交   │ → │ 保管员   │ → │ 运输   │ → │ 班组长  │
│         │    │          │    │ 已确认   │    │ 已通知 │    │ 最终确认│
└─────────┘    └──────────┘    └──────────┘    └────────┘    └─────────┘
    ↓              ↓              ↓              ↓              ↓
 班组长         班组长          保管员         保管员          班组长
 创建订单      提交订单        确认工装       发送通知        确认收到
                                                        ↓
                                                    已完成 ✓
```

### 入库流程

```
┌─────────┐    ┌──────────┐    ┌──────────┐    ┌────────┐    ┌─────────┐
│ 草稿    │ → │ 已提交   │ → │ 保管员   │ → │ 运输   │ → │ 保管员  │
│         │    │          │    │ 已确认   │    │ 已通知 │    │ 最终确认│
└─────────┘    └──────────┘    └──────────┘    └────────┘    └─────────┘
    ↓              ↓              ↓              ↓              ↓
 班组长         班组长          保管员         保管员          保管员
 创建订单      提交订单        确认工装       发送通知        确认入库
                                                        ↓
                                                    已完成 ✓
```

## 订单状态说明

| 状态 | 中文 | 说明 |
|------|------|------|
| `draft` | 草稿 | 订单刚创建，未提交 |
| `submitted` | 已提交 | 已提交等待保管员确认 |
| `partially_confirmed` | 部分确认 | 部分工装已确认 |
| `keeper_confirmed` | 保管员已确认 | 保管员确认通过 |
| `transport_notified` | 运输已通知 | 运输通知已发送 |
| `transport_in_progress` | 运输中 | 运输正在执行 |
| `final_confirmation_pending` | 待最终确认 | 等待最终确认 |
| `completed` | 已完成 | 订单已完成 |
| `rejected` | 已拒绝 | 订单被拒绝 |
| `cancelled` | 已取消 | 订单已取消 |

## 常见操作指南

### 创建订单

1. 点击"创建订单"按钮
2. 选择订单类型（出库/入库）
3. 搜索并添加工装
4. 填写相关信息
5. 点击"提交"

### 提交订单

::: warning 操作确认
提交后将进入保管员审核流程，请确认信息无误。
:::

### 保管员确认

1. 在"待处理"列表查看新订单
2. 点击订单查看详情
3. 确认工装位置和状态
4. 点击"确认"或"拒绝"

### 最终确认

- **出库**: 由班组长确认收到后完成
- **入库**: 由保管员确认已入库后完成
