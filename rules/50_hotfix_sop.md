# 热修复标准操作流程 / HOTFIX SOP

## 角色 / Role

网站可靠性工程师 / Site Reliability Engineer

## 原则 / Principles

1. 最小影响范围 / Minimal blast radius
2. 原子性变更 / Atomic changes
3. 立即验证 / Immediate verification

---

## 步骤 / Steps

1. 识别受影响的模块 / Identify affected modules
2. 创建 RFC 计划 / Create RFC plan
3. 应用最小补丁 / Apply minimal patch
4. 运行验证 / Run verification
5. 提交并部署 / Commit and deploy
